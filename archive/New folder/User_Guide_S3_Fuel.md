# Section S3 (Tunel Kvanj) Fuel Reporting Guide

This guide outlines the mandatory monthly fuel reporting procedure for Section S3 to ensure integration with the project-wide Fleet Management system.

## 1. Reporting Requirements
All fuel consumption at the **Tunel Kvanj** site must be recorded daily using the provided template.

### Frequency
*   **Daily**: Operators/Site Managers must record each fueling event.
*   **Monthly**: The completed file must be sent to the Mostar Central Office by the **5th of the following month**.

## 2. Using the Template
The [S3_Fuel_Reporting_Template.xlsx](file:///home/k/Downloads/azvirt_dms/archive/New%20folder/S3_Fuel_Reporting_Template_September_2025.xlsx) contains the following fields:

| Field | Description | Requirement |
| :--- | :--- | :--- |
| **Date** | Date of fueling (DD.MM.YYYY) | Mandatory |
| **Machine ID** | Inventar code (e.g., OTP-1388) | Must match Reference list |
| **Quantity** | Liters of fuel received | Numeric only |
| **Odometer** | Current km or motorhour reading | Critical for efficiency analysis |
| **Received By** | Name of the person responsible | Signature equivalent |

## 3. Submission Workflow
1.  Open the latest monthly template.
2.  Input fueling data as it occurs.
3.  Cross-check total monthly quantity against site tank deliveries (e.g., Hifa/Petrol invoices).
4.  Email the file to `audit@azvirt-dms.com` with the subject: `[S3] Fuel Report - [Month] [Year]`.

> [!IMPORTANT]
> Failure to report fuel consumption for machines with recorded maintenance (oil services) will be flagged as a data discrepancy in the monthly audit.
